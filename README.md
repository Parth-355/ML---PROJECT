# Hate Speech Detection using Machine Learning

## 📌 Project Overview

This project detects toxic or hate speech comments using NLP and Machine Learning.

## 🚀 Features

* Text Cleaning
* TF-IDF Vectorization
* Logistic Regression Model
* Toxic Comment Detection

## 🛠️ Technologies Used

* Python
* Scikit-learn
* NLTK
* Google Colab

## ▶️ How to Run

1. Open the notebook in Google Colab
2. Run all cells
3. Test with your own input

## 📊 Example

"I hate you" → Toxic ❌
"Great job!" → Not Toxic ✅
